# GoalGlider02 GitHub Version

This is the frontend-only version of the Simon game, designed for GitHub Pages deployment.

## Included
- React Components (`src/`)
- HTML and CSS (`public/`)
- `simon-info.html` with project overview

## How to Deploy
1. Push to a GitHub repository.
2. Enable GitHub Pages in repo settings.
3. Set the root to the branch and `/public` folder.
4. Access your deployed game via the GitHub Pages link.

Enjoy playing Simon!
